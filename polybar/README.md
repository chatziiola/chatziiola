# Polybar

My polybar configuration files can be organized and analyzed carefully in the following categories. This is the result of my *need* to change a lot of stuff quite often, while maintaining the functionality **AND** the speed at which I could turn an idea into reality. So without further ado:
- [Skins](#skins), these are the *carefully (!) named* `.ini` files
- [Scripts](#scripts), these are what makes my polybar setup function the way I like it to be 
- [Modules](#modules), these are the same for every skin and are defined in `modules.ini`
- [Colors](#colors), the colors constantly updated when [pywal](TODO) is run
